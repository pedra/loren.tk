Loren Ipsum Generator

Commands & options:

Usage : http://loren.tk/[COMAND]/[OPTION]/[OPTION]

Word

loren.tk/w/qtd

qtd : quantidade de palavras.

Ex.: loren.tk/w/10

Charters

loren.tk/c/min/max

min : quantidade mínima de carecters nas palavras.

max : quantidade máxima de carecters nas palavras.

Ex.: loren.tk/c/3/20

Paragraphs

loren.tk/p/qtd/tag

qtd : quantidade de parágrafos.

tag : se presente, usa a tag <p/> entre os parágrafos (1/0).

Ex.: loren.tk/p/3

Ex.: loren.tk/p/3/1

Randomic

loren.tk/r/qtd

qtd : quantidade de caracteres.

Gera uma palavra com caracteres aleatórios.

Ex.: loren.tk/r

Ex.: loren.tk/r/8

Examples

HTML iframe:

<iframe src="http://loren.tk/p/4/1"></iframe>

Jquery Ajax

in HTML
<div id="ajax"> --- </div>

in Javascript
$("#ajax").load("http://loren.tk/p/2/1");

PHP Codes

<?php
$loren = file_get_contents('http://loren.tk/p/10/1');

//Display 10 paragraphs with '<p>' tags.
echo $loren;
